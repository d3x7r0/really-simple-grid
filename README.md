# Really Simple Grid

A really simple (mostly) drop-in replacement for Bootstrap grids

## Getting Started
Add to your project using Bower:

`bower install really-simple-grid`

Or NPM:

`npm install really-simple-grid`

## Usage

```HTML
<div class="container">
    <div class="row">
        <div class="col-4"></div>
        <div class="col-4 col-offset-2"></div>
        <div class="col-2"></div>
    </div>
</div>
```
